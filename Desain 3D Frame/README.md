# 🚁 Desain 3D Frame Drone Menggunakan Autodesk Inventor

## Deskripsi

Project ini merupakan perancangan model 3D sebuah frame drone tipe quadcopter menggunakan Autodesk Inventor. Desain dibuat sebagai media pembelajaran untuk memahami proses Computer-Aided Design (CAD), mulai dari pembuatan part, perakitan (assembly), hingga visualisasi model 3D.

Project ini juga bertujuan untuk meningkatkan kemampuan dalam mendesain komponen mekanik, memahami hubungan antar komponen, serta menghasilkan model yang siap digunakan sebagai acuan proses manufaktur maupun pengembangan lebih lanjut.

---

## Tujuan Project

- Mempelajari proses desain 3D menggunakan Autodesk Inventor.
- Membuat model frame drone tipe quadcopter.
- Melatih kemampuan dalam pembuatan part dan assembly.
- Memahami konsep desain produk dan visualisasi 3D.
- Menyusun dokumentasi sebagai portofolio engineering.

## Hasil Project

Model 3D berhasil dirancang menggunakan Autodesk Inventor dengan memperhatikan struktur frame, posisi lengan (arm), serta area pemasangan komponen seperti motor, flight controller, dan baterai. Desain ini menjadi latihan dalam penerapan konsep CAD dan pengembangan produk berbasis desain mekanik.
